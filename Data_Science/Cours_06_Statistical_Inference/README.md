# Content of this folder

This folder gathers the exercices and projects relative to the Course 6 of Data Science Specialization : ***Statistical Inference***.

This folder is divided in 4 subfolders. Any of them covers one specific point of the regression models :

 1. <span style="color:blue">Semaine_01</span> : Probability & Expected Values
 2. <span style="color:blue">Semaine_02</span> : Variability, Distribution, & Asymptotics
 3. <span style="color:blue">Semaine_03</span> : Intervals, Testing, & Pvalues
 4. <span style="color:blue">Semaine_04</span> : Power, Bootstrapping, & Permutation Tests
